# Admin Scaffold

This package is a Week 1 placeholder boundary for the future Admin UI (PRD
Section 6, FINAL_ALIGNMENT Week 4).

## Scope

- Week 1: package boundary only. No routes, no CRUD, no auth.
- Week 4: RBAC, CSRF, idle timeout, API auth, SQL injection defence, audit log.

## What does NOT belong here yet

- Admin UI pages or API routes.
- Authentication or RBAC middleware.
- Supabase schema or RLS policies.
- Production secrets or service keys.
- Scenario editing or KB management endpoints.

All of the above require their respective Week blocker source data before
implementation.
